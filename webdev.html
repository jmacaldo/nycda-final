Web Development


Certainly a work in progress! The two screenshots on the right show webpages that demonstrate the concepts learned in class so far: HTML, CSS, and Bootstrap. As you can see, the two pages do look quite similar. Minimal design elements and color are primarily used. I except to venture out into using more color and elaborate design as my skills as a web developer continue to grow!

“Inspiration Page” is the first assignment for class. This very basic HTML page serves as a repository of aesthetic ideas that inspire the student. Some basic CSS is also used here to style the navbar and main containing elements of the page.

“About Me” shows some more HTML skills and a bit more advanced CSS. The page features an unusual CSS workaround trick wherein a div is rotated 90 degrees and the text inside it is rotated back so it’ll appear right side up. The result is a horizontal scrolling element. It’s clunky for sure, but it’s a novel solution to a problem!
